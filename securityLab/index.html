Security Lab

Il Security Lab è una macchina virtuale che permette agli studenti di svolgere le esercitazioni del corso di Sicurezza. La macchina virtuale è stata creata con Virtual Box e può essere importata seguendo le istruzioni disponibili a questo indirizzo.

È possible scaricare la macchina virtuale al seguente indirizzo

La password per accedere alla macchina virtuale è securitylab

Nota Dopo aver scaricato la macchina virutale aprire il terminale e eseguire i seguenti comandi:

  • sudo chown -R www-data:www-data /var/www/html
  • sudo chmod -R 754 /var/www/html/

Aggiornamento del 23 Aprile 2020: Rilasciata la versione 1.1 scaricabile a questo indirizzo. Importando questa versione non sarà più necessario eseguire i comandi indicati sopra. Questa versione apporta delle modifiche alla esercitazione 3.
